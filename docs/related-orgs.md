# Related organizations

## thermal-label

**[thermal-label](https://github.com/thermal-label)** maintains the **TypeScript drivers** and **transport** layer Burnmark uses at print time — USB, TCP, WebUSB, Web Bluetooth, WebHID depending on device.

- Website: [https://thermal-label.github.io/](https://thermal-label.github.io/)

## mbtech-nl

**[mbtech-nl](https://github.com/mbtech-nl)** hosts small shared libraries such as **`@mbtech-nl/bitmap`**, which keeps monochrome renders consistent between design tools and printer drivers.

- Website: [https://mbtech-nl.github.io/](https://mbtech-nl.github.io/)
